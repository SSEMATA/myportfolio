How We Built a Smart Traffic Light System

This article walks you through how we used Arduino, sensors, and cardboard models...

To address the growing issue of traffic congestion in urban areas, we developed a Smart Traffic Light System using an Arduino board and ultrasonic sensors. This system replaces traditional timer-based lights with a dynamic, sensor-driven approach that adapts in real-time to the presence of vehicles. Four ultrasonic sensors are installed at a four-way junction‚Äîeach detecting vehicle proximity and sending signals to the Arduino, which then prioritizes traffic flow by turning green lights on only where needed while managing red and yellow lights accordingly. This improves efficiency, reduces waiting time, and minimizes fuel waste. We built the prototype using a wooden base and cardboard stands, equipped with LEDs for each signal color. The system also allows for expansion to include pedestrian safety signals, emergency vehicle detection, and buzzers for alerting signal changes. Prior to the physical setup, we created a full simulation using Tinkercad to ensure accurate logic and functionality. This project demonstrates how affordable microcontrollers and sensors can be combined to create intelligent, scalable traffic solutions for smarter cities.
